# QR Code Order System (Next.js)

Dự án này sử dụng Next.js (TypeScript, App Router, Tailwind, ESLint) để xây dựng hệ thống đặt hàng bằng mã QR.

## Chức năng dự kiến
- Đăng nhập, đăng ký
- Dashboard
- Quản lý người dùng
- Quản lý sản phẩm
- Quản lý hóa đơn

## Khởi động dự án
```bash
npm run dev
```

Truy cập [http://localhost:3000](http://localhost:3000) để xem giao diện.

## Cấu trúc mặc định
- src/app: Routing chính
- src/components: Các component tái sử dụng
- src/styles: Style chung (Tailwind)
